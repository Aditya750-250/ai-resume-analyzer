# 📄 AI Resume Analyzer

An AI-powered resume analyzer built using Python, Streamlit, and Gemini AI.

## Features
- Resume PDF Upload
- AI Resume Analysis
- ATS Score Suggestions
- Skill Recommendations
- Resume Improvement Tips

## Tech Stack
- Python
- Streamlit
- Gemini AI
- PyPDF2

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
